# 🍄 超级马力欧 (Super Mario)

基于 HTML5 Canvas 开发的经典超级马力欧游戏，完整还原原版游戏的核心玩法和体验。

## ✨ 功能特性

### 🎮 核心玩法
- ✅ 经典平台跳跃机制，物理引擎还原原版手感
- ✅ 三种角色状态：小马力欧、大马力欧、火焰马力欧
- ✅ 经典道具系统：超级蘑菇、火焰花、无敌星、1UP蘑菇
- ✅ 敌人系统：蘑菇怪(Goomba)、乌龟(Koopa)
- ✅ 关卡系统：难度平滑递增的随机生成关卡
- ✅ 碰撞系统：精确的像素级碰撞检测
- ✅ 音效系统：Web Audio API 生成经典游戏音效

### 🎨 游戏特色
- ✅ 完整的开始界面，包含操作说明、道具介绍、得分规则
- ✅ 自适应全屏显示，支持任意窗口大小
- ✅ 像素风格渲染优化，画面清晰锐利
- ✅ 经典爬旗杆动画，完美还原原版体验
- ✅ 本地存储最高分记录
- ✅ 流畅的60fps游戏体验

### ⌨️ 操作方式
| 操作 | 按键 |
|------|------|
| 左右移动 | A / D 或 方向键 ← / → |
| 跳跃 | 空格键 / W / 方向键 ↑ |
| 下蹲 | S / 方向键 ↓ (大马力欧状态) |
| 冲刺 | Shift 键 |
| 发射火球 | J 键 (火焰马力欧状态) |
| 暂停游戏 | P 键 |
| 开关音效 | M 键 |
| 开始游戏 | 回车键 |

## 🚀 快速开始

### 方式一：直接打开
直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 方式二：本地服务器（推荐）
```bash
# 使用 Python 启动简单服务器
python3 -m http.server 8080

# 或使用 Node.js
npx serve .
```
然后在浏览器访问 `http://localhost:8080`

## 🎯 游戏规则

### 道具效果
| 道具 | 效果 |
|------|------|
| 🍄 超级蘑菇 | 马力欧变大，可以承受一次伤害 |
| 🌼 火焰花 | 获得发射火球能力，可以远程攻击敌人 |
| ⭐ 无敌星 | 10秒无敌状态，可以撞飞所有敌人 |
| 🍀 1UP蘑菇 | 增加一条生命值 |

### 得分规则
| 行为 | 得分 |
|------|------|
| 收集金币 | 200分 |
| 踩踏敌人 | 100分 |
| 获得道具 | 1000分 |
| 通关奖励 | 5000分 |

### 生命值
- 初始3条生命
- 小马力欧受到伤害直接损失一条生命
- 大/火焰马力欧受到伤害会变小
- 跌落屏幕直接损失一条生命
- 生命耗尽游戏结束

## 🎮 关卡设计

游戏采用渐进式难度设计：
- **第1关**: 平坦地形，少量敌人，熟悉操作
- **第2关**: 增加少量平台和沟壑
- **第3关**: 加入管道和乌龟敌人
- **第4-5关**: 逐步增加难度，更宽的沟壑和更多敌人
- **第5关后**: 难度保持稳定，提供持续挑战

关卡自动生成机制保证每一次游戏体验都不同，同时确保所有关卡都可以正常通关，没有死路。

## 🤝 开发说明

### 代码结构
- `game.js` 采用模块化设计，功能独立：
  - 音频系统 (1-180行)
  - 配置常量 (181-300行)
  - 游戏状态管理
  - 输入事件处理
  - 关卡生成逻辑
  - 物理引擎和碰撞检测
  - 游戏对象更新逻辑
  - 渲染系统

### 扩展开发
可以很方便地添加新功能：
- 新增敌人类型：在 `ENEMY_TYPES` 添加类型，实现对应的更新和渲染逻辑
- 新增道具：在 `POWERUP_TYPES` 添加类型，实现效果逻辑
- 新增关卡元素：在 `TILE_TYPES` 添加类型，实现碰撞和渲染
